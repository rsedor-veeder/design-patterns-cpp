
# Command-Line Build

$ cd into the directory

[source, cmake]
----
$cd solid-cpp
----

Run cmake against the CMakeLists.txt.  Assume this is *nix environment and case sensitive.

[source, cmake]
----
$cmake CMakeLists.txt
----

When it's done, run make to build the executables.
[source, cmake]
----
$make
----
